## 02. The signature curve

The PROCERTUS logotype contains two interlocking checkmarks. Their geometry is not
a circle and not a square; it is a compound curve with one deep sweep and one sharp
return. We extracted this DNA and encoded it as an asymmetric radius that can be
applied to any interactive component.

## 03. Kinetic pairing

Not all actions move in the same direction. Some push the user forward (Submit,
Confirm, Save). Others pull them back or offer an exit (Cancel, Back, Dismiss).
The Signature Curve encodes this distinction through its orientation.

## 04. The pulse of verification

A PROCERTUS button does not simply change color on press. It travels. On active,
the button translates <strong>1px</strong> downward with zero duration and simultaneously fires a
beacon pulse: a ring of light that expands outward from the button's edge over
<strong>800ms</strong> before fading to nothing. This is the command confirm interaction model.
