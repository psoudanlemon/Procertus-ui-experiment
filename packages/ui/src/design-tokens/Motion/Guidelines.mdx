## 01. Principles

Three rules govern every animation in the platform. They exist to keep motion
precise, confident, and invisible when it should be.

<div className="grid grid-cols-1 md:grid-cols-3 gap-6 mt-6">
  <div className="rounded-xl border-2 border-brand-primary-200 dark:border-brand-primary-800 bg-card p-6">
    <h4 className="font-sans text-base font-bold text-foreground m-0 mb-3">Precise, not playful</h4>
    <p className="text-sm text-muted-foreground leading-[1.6] m-0">
      This is a certification platform. Motion should feel controlled and
      confident. Short durations, no bounce, no spring physics. If a user
      notices the animation itself, it is too much.
    </p>
  </div>
  <div className="rounded-xl border-2 border-brand-primary-200 dark:border-brand-primary-800 bg-card p-6">
    <h4 className="font-sans text-base font-bold text-foreground m-0 mb-3">Fade + scale to appear, slide to navigate</h4>
    <p className="text-sm text-muted-foreground leading-[1.6] m-0">
      Floating UI (popovers, dialogs, menus) fades in with a subtle scale.
      Lateral content (sheets, navigation panels) slides. This distinction
      separates "new layer" from "adjacent content."
    </p>
  </div>
  <div className="rounded-xl border-2 border-brand-primary-200 dark:border-brand-primary-800 bg-card p-6">
    <h4 className="font-sans text-base font-bold text-foreground m-0 mb-3">Respect the user</h4>
    <p className="text-sm text-muted-foreground leading-[1.6] m-0">
      When <code className="text-xs bg-muted px-1 py-0.5 rounded">prefers-reduced-motion</code> is
      active, all animations and transitions are disabled globally. No
      component-level opt-outs, no exceptions.
    </p>
  </div>
</div>

## 02. Easing curves

Two curves handle the entire system. Fewer curves means more consistency;
more consistency means motion becomes invisible infrastructure rather than
noticeable style.

<div className="grid grid-cols-1 md:grid-cols-2 gap-6 mt-6">

## 03. Animation patterns

Every animation in the system maps to one of four patterns. Knowing which
pattern applies to a given component eliminates guesswork.

<div className="grid grid-cols-1 md:grid-cols-2 gap-6 mt-6">
  <div className="rounded-xl border border-border bg-card p-6">
    <span className="font-semibold text-foreground block" style={{ fontSize: '1.5rem' }}>Fade + scale</span>
    <span className="uppercase block mb-3" style={{ fontSize: '12px', fontWeight: 600, letterSpacing: '0.1em', color: 'var(--muted-foreground)', marginTop: '8px' }}>Floating layers</span>
    <p className="text-sm text-muted-foreground m-0" style={{ lineHeight: 1.6 }}>
      Opacity from 0 to 1 combined with scale from 95% to 100%. Communicates
      "a new layer appeared above the current surface." Used for all floating UI
      that originates from a trigger point.
    </p>
  </div>
  <div className="rounded-xl border border-border bg-card p-6">
    <span className="font-semibold text-foreground block" style={{ fontSize: '1.5rem' }}>Slide</span>
    <span className="uppercase block mb-3" style={{ fontSize: '12px', fontWeight: 600, letterSpacing: '0.1em', color: 'var(--muted-foreground)', marginTop: '8px' }}>Lateral content</span>
    <p className="text-sm text-muted-foreground m-0" style={{ lineHeight: 1.6 }}>
      Translate along one axis combined with fade. Communicates "content is
      arriving from, or departing to, a specific direction." Direction follows
      the element's position: sheets slide from their edge, navigation content
      slides in from its origin side.
    </p>
  </div>
  <div className="rounded-xl border border-border bg-card p-6">
    <span className="font-semibold text-foreground block" style={{ fontSize: '1.5rem' }}>Expand / collapse</span>
    <span className="uppercase block mb-3" style={{ fontSize: '12px', fontWeight: 600, letterSpacing: '0.1em', color: 'var(--muted-foreground)', marginTop: '8px' }}>Content reveal</span>
    <p className="text-sm text-muted-foreground m-0" style={{ lineHeight: 1.6 }}>
      Height animation from 0 to auto (via Radix CSS variables). Communicates
      "more content exists here and is now visible." The surrounding layout
      adjusts smoothly as the element grows or shrinks.
    </p>
  </div>
  <div className="rounded-xl border border-border bg-card p-6">
    <span className="font-semibold text-foreground block" style={{ fontSize: '1.5rem' }}>Continuous loop</span>
    <span className="uppercase block mb-3" style={{ fontSize: '12px', fontWeight: 600, letterSpacing: '0.1em', color: 'var(--muted-foreground)', marginTop: '8px' }}>Ambient state</span>
    <p className="text-sm text-muted-foreground m-0" style={{ lineHeight: 1.6 }}>
      Infinite iteration with linear or ease-out timing. Communicates either
      "work is happening" (spinner, skeleton pulse) or "this element is
      exceptional" (glow-spin). Never tied to a discrete user action.
    </p>
  </div>
</div>

## 04. Branded motion

Two custom animations exist outside the standard pattern vocabulary. They
carry brand identity and should not be replicated or extended without
deliberate design intent.

<div className="grid grid-cols-1 md:grid-cols-2 gap-6 mt-6">
  <div className="rounded-xl border-2 border-brand-primary-200 dark:border-brand-primary-800 bg-card p-6">
    <span className="font-semibold text-foreground block" style={{ fontSize: '1.5rem' }}>Command pulse</span>
    <span className="uppercase block mb-3" style={{ fontSize: '12px', fontWeight: 600, letterSpacing: '0.1em', color: 'var(--muted-foreground)', marginTop: '8px' }}>Tactile feedback</span>
    <p className="text-sm text-muted-foreground m-0" style={{ lineHeight: 1.6 }}>
      An 800ms box-shadow expansion that radiates outward from a pressed button.
      It transforms a click from a binary state change into a physical gesture:
      the user pushed, and the interface responded with weight. Two variants
      exist: default (ring color) and destructive (destructive color).
    </p>
  </div>
  <div className="rounded-xl border-2 border-brand-accent-200 dark:border-brand-accent-800 bg-card p-6">
    <span className="font-semibold text-foreground block" style={{ fontSize: '1.5rem' }}>Glow-spin</span>
    <span className="uppercase block mb-3" style={{ fontSize: '12px', fontWeight: 600, letterSpacing: '0.1em', color: 'var(--muted-foreground)', marginTop: '8px' }}>Visual gravity</span>
    <p className="text-sm text-muted-foreground m-0" style={{ lineHeight: 1.6 }}>
      A 6-second rotating conic gradient through two counter-rotating blur
      layers. Reserved for the L1 Standout elevation tier. One per viewport at
      most. Freezes at 135 degrees when
      <code className="text-xs bg-muted px-1 py-0.5 rounded">prefers-reduced-motion</code> is
      active.
    </p>
  </div>
</div>
