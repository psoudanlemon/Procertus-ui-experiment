# CertificationIntentPicker

Maps the three `meta.wizard.entryPoints` to **`SelectChoiceCard`**: **regulated** product path (primary), **ATG** (secondary), **innovation/SSD** (tertiary). `defaultCertificationIntentOptionsEn` is stub copy for Storybook; apps should supply translated options.

- Pair with a container step (e.g. `StepLayout`) and parent state for the wizard. No `fetch` or `react-router` here (Task D).
